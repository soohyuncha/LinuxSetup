## LinuxSetup
Linux setup of oh-my-zsh, tmux, and vim

### 1) oh-my-zsh
```console
./zsh_setup.sh
```

### 2) tmux
```console
./tmux_setup.sh
```

### 3) vim
```console
./vim_setup.sh
```

Note that in each ***setup.sh*** file, there is a commented command. If you need it, then uncomment this command or manually enter this command in your shell.
